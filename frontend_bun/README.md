# Run the project
```bash
# Clone the repository
git clone https://github.com/anamulislamshamim/simple-gmail-ai-agent.git
# create .env file and added 
VITE_BASE_URL=http://localhost:3000
# Go to project directory
cd ./simple-gmail-ai-agent/frontend_bun
# Install Dependencies
bun install
# Run the app
bun dev
```
